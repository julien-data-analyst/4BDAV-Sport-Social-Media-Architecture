# Services Worker notif
Ce service permet la création d'un worker sous Python 
qui notifie la personne de nouveaux posts des personnes qui follows.
Ce worker permet d'envoyer une requête HTTP via websocket en utilisant Redis pour identifier les nouveaux posts facilements
