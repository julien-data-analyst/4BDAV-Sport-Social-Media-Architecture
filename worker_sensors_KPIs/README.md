# Services Worker sensors KPIS
Ce service permet la création d'un worker sous Python 
qui permets le calcul et l'insertion des KPIS et des activités associées.
Ce worker permettra de faire la connexion entre la base de données **InfluxDB** et la base de données **PostgreSQL**
en surveillant si l'activité est terminée ou non et appliquer dans le cas échéant les calculs de KPIs qui seront 
insérées ensuite dans la base de données. 

Pour l'instant rien n'a été développée dû au manque de temps.
