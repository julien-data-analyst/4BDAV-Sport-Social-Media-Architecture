# Service [Cloudinary](https://cloudinary.com/)

Ce service va nous servir principalement dans le téléversement, le téléchargement
 de médias afin de ne pas complexifier le stockage de ce type de données.
Cela sera très utile dans la gestion des médias pour le réseau social sportif.
C'est un service cloud qui nécessitera juste la création d'une clé.
